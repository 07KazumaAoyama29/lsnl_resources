## 輪講とは
### 輪講の目的<br>
- 輪講を通じて、下記のシラバスの目標を達成しつつ、**異なる研究グループ**間の親睦を深める<br>
- 効率の良い学習を実現する<br>
->学習コストと学習利益の観点から、各自が学習した内容を互いに発表し合うと**1人分の学習コストから参加人数分の学習利益**を得られる。これを目指す。<br>
詳しくは、少し硬い文章ですが、redmineの以下のページを参照してください。<br>
https://rm.lsnl.jp/projects/adm/wiki/%E5%8B%89%E5%BC%B7%E4%BC%9A%E8%BC%AA%E8%AC%9B%E3%81%AE%E5%8E%9F%E7%90%86<br>
### 授業目的 / Course Purpose<br>
- (**学部**)<br>
情報科学分野の文献から、**情報ネットワーク**に関する適当な題材を選び、輪講形式でその内容を他の受講者に紹介する。<br>
文献に含まれている情報をよく**理解し**、**正確に把握する**と同時に、**理解した内容を要領よくまとめて発表**し、さらに**発表された内容をもとに有効な討論**を行うことを目的とする。<br>
「卒業実験及び演習」及び「外国書講読」を含めた卒業研究活動の一環として行う。**必ず出席することが必要**である。<br>
- (**大学院**)<br>
各自の研究テーマに関連する書籍や論文を輪読することにより、情報ネットワーク学に関する**最新技術を理解する**とともに、<br>
**英文読解力**および**コミュニケーションスキル (プレゼンテーション技術・テクニカルライティング技術)**を習得する。<br>
「文献演習」、「特別実験及び演習」、「特別研究」を含めた研究活動の一環として行う。<br>

### 到達目標 / Learning Goals	<br>
- (**学部**)<br>
情報ネットワークに関する**学部レベルの教科書**が読めるようになる。<br>
情報ネットワークに関する**学術論文の要旨を理解できる**ようになる。<br>
- (**大学院**)<br>
情報ネットワークに関する**大学院レベルの英語教科書**が読めるようになる。<br>
情報ネットワークに関する**学術論文が読める**ようになる。<br>

### 授業方法 / Course Format	<br>
- (**学部**)<br>
輪講形式で実施する<br>
- (**大学院**)<br>
各自の研究テーマに関連する書籍や論文の内容をまとめ、発表する。<br>
その内容ついて、教員やグループメンバらとのディスカッションを行うことで理解を深める。<br>

## 実施方式の決定(提案A「個人発表形式」 vs 提案B「グループ発表形式」)
輪講の実施方式について、私自身輪講をしたことがないので皆さんと話し合って決めたいです。<br>
私が考えているのは以下の二つです。どちらであってもスライドを作成し、それを基に発表してもらいます。
- A: <br>
発表形式: 一人一本の論文の内容を発表する。<br>
質疑応答: 発表者が質問を募り、それに解答する<br>
メリット: なんか輪講っぽい<br>
デメリット: 論文の読む本数が少ない(一人一本)、個人で発表するのでさみしい、質疑応答で毎回同じ人が質問する<br>
- B: <br>
発表形式: 大崎先生の大学院の授業みたいに、3-4人でグループを組んでグラフ系の論文と通信系の論文を一本ずつ発表する。<br>
質疑応答: 3-4グループに分かれ、そこで議論を行う。1グループに一人発表者が必ずいるようにする。<br>
メリット・・・異なる研究グループのメンバと親睦を深めることができる、楽しそう、発表準備の時間が取れる<br>
デメリット・・・特になし？<br>

### 発表する論文の選定基準
グループで行うのであれば、グラフ系の論文と通信系の論文を一つずつ。<br>
ただし、論文はトップカンファレンスのもの。発表年は問わない。<br>
ジャーナルの場合は分野さえ合っていれば雑誌名は問わない。<br>
書籍の場合も分野さえ合っていれば可。<br>

## 輪講のルール
- 時間厳守<br>
研究室で何かイベントをすると、時間を押してしまうことが多いのですが、輪講は**時間厳守**で行きます。<br>
１コマ(100分)丸々やるのではなくて、60分強ぐらいで終わる感じで行きたいなと思っています(ボリュームが少なすぎてもダメですが、多すぎてもダメです)。<br>
- 発表スライドの共有<br>
輪講が始まるまでに、redmineの開催チケットにスライドを添付してください。<br>
(小川さんも見れるように、メールでも送信してください)<br>
- 発表準備について<br>
発表準備は出席しなくても大丈夫ですが、次の週に発表の人は必ず出席してください。<br>
